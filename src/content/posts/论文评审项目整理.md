快速上手创建Vue应用
```
    npm create vue@latest 项目名称
```

Vue项目架构
```
-- frontend
    -- .vscode        // vscode的项目级配置文件，调试配置，推荐插件
    -- dist           // 运行npm run build 之后生成,里面是压缩后的HTML/CSS/JS，可以拿去部署到服务器
    -- node_modules     // 依赖包安装目录，运行npm install后生成，里面是项目中所有第三方库
    -- public           // 静态资源拷贝，放进去的文件会在构建时原封不动的拷贝到最终输出
    -- src              // Vue组件，页面，路由，状态管理，样式都在这里
    -- .gitignore       // 告诉Git哪些文件不要提交
    -- index.html       // Vite项目的页面入口模板，从这里加载src/main.ts/js
    -- jsconfig.json    // 编辑器/工具用的配置（JS项目用，路径别名提示，智能补全）
    -- package.json     // 项目说明书，记录项目名称，依赖，脚本命令
    -- package-lock.json    // 锁定依赖的具体版本
    -- README.md        // 项目说明文档
    -- vite.config.js   // Vite的配置文件，端口、代理、路径别名、打包配置、插件都在这里
```
